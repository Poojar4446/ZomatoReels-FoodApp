# ZomatoReels FoodApp - Frontend

React + Vite frontend for the ZomatoReels Food Delivery Platform.

## Features
- User Authentication (Login/Register)
- Food Reels Feed (TikTok-style video feed)
- Food Partner Dashboard
- Responsive Design
- Modern UI/UX

## Technologies
- React.js
- Vite
- React Router
- Tailwind CSS (or your CSS framework)
- Axios/Fetch for API calls

## Setup
```bash
npm install
npm run dev
